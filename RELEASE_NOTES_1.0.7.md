# NocLauncher v1.0.7

## Что нового
- Полностью обновлён каркас окна лаунчера:
  - реальное скругление окна;
  - прозрачные внешние края (за пределами формы);
  - frameless-режим с более аккуратным визуалом.
- Добавлены верхние кнопки управления окном:
  - свернуть;
  - закрыть.
- Splash-окно переработано под единый стиль лаунчера:
  - скруглённая форма;
  - прозрачный внешний фон;
  - размер уменьшен примерно на 40%.
- Снижено затемнение фона splash примерно на 60% (более светлый и читаемый экран загрузки).
- Полностью удалён анимированный canvas-фон из лаунчера и удалена настройка его переключения.
- Улучшен Bedrock launch-flow:
  - после успешного старта Minecraft Bedrock лаунчер скрывается сразу.

## Важно (Windows Defender / SmartScreen)
При первом запуске Windows может показывать предупреждение SmartScreen/Defender.
Это возможно из-за отсутствия коммерческой цифровой подписи у сборки, а не обязательно из-за вредоносного кода.

Рекомендуется:
1. Скачивать сборки только из официального GitHub Releases.
2. Проверять SHA256-хэш опубликованных файлов.
3. При необходимости дополнительно проверять файл в VirusTotal.
