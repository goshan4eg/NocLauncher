# NocLauncher SEO / Discoverability Checklist

## GitHub Repository Settings (About block)
Set these values in repository **About**:

- **Description:** `NocLauncher — официальный лаунчер Minecraft Java + Bedrock (Windows)`
- **Website:** `https://github.com/NocCorporation/NocLauncher/releases`
- **Topics:** `noclauncher`, `minecraft`, `minecraft-launcher`, `bedrock`, `java`, `electron`, `windows`

## Release Hygiene
- Keep version tags consistent (`v1.0.6`, `v1.0.7`, ...)
- Every release should include:
  - installer (`.exe`)
  - checksum file (`SHA256SUMS-*.txt`)
  - defender/smartscreen note

## Public Naming Consistency
Use the same canonical name everywhere:
- `NocLauncher`
- `NocCorporation/NocLauncher`
- `Official GitHub Releases`

## Recommended Search Phrases
- `NocLauncher GitHub`
- `NocCorporation NocLauncher`
- `NocLauncher Minecraft Java Bedrock`

## Anti-Fake Rule
In posts/channels always include:
- direct link to repo
- direct link to latest releases
- short note: "Скачивайте только с официального GitHub Releases"
