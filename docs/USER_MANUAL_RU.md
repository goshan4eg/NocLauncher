# NocLauncher — руководство пользователя

## 1. Режимы

- **JAVA** — Java Edition, поддержка профилей Vanilla/Forge/OptiFine.
- **BEDROCK** — Bedrock режим запуска.

## 2. Профили и запуск

### Vanilla
Используйте для максимальной совместимости и диагностики базы.

### Forge
Используйте для модов. Проверяйте совместимость версии модов с версией игры.

### OptiFine
Используйте для повышения FPS и графических настроек.

## 3. Производительность

Рекомендуемая базовая конфигурация:
- FPS Boost Mode: **ON**
- FPS preset: **balanced**
- Low-power UI: **ON**

Это даёт хороший баланс стабильности/производительности на большинстве ПК.

## 4. Поведение лаунчера при старте игры

Лаунчер уходит в фон при старте Minecraft и возвращается после выхода из игры.

## 5. Оффлайн-скины

Поддерживаются источники:
- локальный файл
- ник
- URL
- авто

Если скин не применился — проверьте формат PNG и логи.

## 6. Серверы

Лаунчер добавляет серверы автоматически в:
- `<gameDir>/servers.dat`
- `%APPDATA%/.minecraft/servers.dat`

## 7. Диагностика

Используйте раздел **Логи** для анализа ошибок запуска.

Для баг-репорта прикладывайте:
- версию лаунчера
- ОС
- версию Minecraft
- профиль запуска
- полный лог
