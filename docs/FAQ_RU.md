# FAQ — NocLauncher

## Будет ли работать без Microsoft-аккаунта?
Да. Текущая версия работает в offline-first режиме.

## Почему лаунчер не закрывается полностью при старте игры?
Это специально: лаунчер уходит в фон, чтобы корректно вернуться после выхода.

## Как уменьшить нагрузку на видеокарту?
Включите low-power UI (отключение тяжёлых визуальных эффектов).

## Что выбрать: Vanilla, Forge или OptiFine?
- Vanilla — стабильный базовый запуск
- Forge — для модов
- OptiFine — для FPS/графики

## Где посмотреть ошибки?
В разделе логов лаунчера.

## Почему macOS ругается на приложение?
Для beta-сборок без notarization это нормально. Открывайте через ПКМ → Open.

## Где скачать файлы?
В GitHub Releases → Assets.
